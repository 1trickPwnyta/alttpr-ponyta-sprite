# How to build
1. Download ALTTPNG from https://github.com/spannerisms/ALttPNG/releases.
2. Open ponyta_sheet.paint in Microsoft Paint (may require Windows 10 or 11?)
3. Enable the layers panel by clicking the button in the top toolbar.
4. Click the icon in the corner of the layer preview for the bottom (background) layer to make it invisible.
5. Go to File > Save as PNG. Confirm when prompted.
6. Open PNGto4BPP.jar from ALTTPNG.
7. Load the PNG file you saved in a previous step.
8. Click Save/Patch to and save the file as ponyta.zspr.
9. For Sprite name, enter Ponyta.
10. Enter your name and a short version of your name.
11. Click Convert to export the ponyta.zspr file.

# How to use
1. Go to https://alttpr.com/en/randomizer and generate a randomized game.
2. On the download page, for "Play As", select "Load Custom Sprite" and upload ponyta.zspr.
3. Click Save ROM.
